# BrokenKeyboard

🇷🇺 Русский
Broken Keyboard — это проект, где клавиши клавиатуры постоянно перемешиваются и ведут себя непредсказуемо.
Идеально подходит для челленджей, тестирования скорости реакции или просто весёлого хаоса на компьютере.
Каждое нажатие клавиши может неожиданно превратиться в другое — готовься к полной неожиданности!


🇬🇧 English
Broken Keyboard is a project where the keyboard keys are constantly shuffled and behave unpredictably.
Perfect for challenges, reaction time tests, or just some fun chaos on your computer.
Every key press might unexpectedly turn into another — get ready for total surprise!
